# Docker Commands
---
## Main Basic Docker COmamands
#### check Docker Version
```
docker -v
docker --version
'''
#### Running Containers Check
```
docker ps
'''
#### List all Container
```
docker ps -a
```
#### Stop a Container
```
docker stop <Cpntainer ID>
```
#### to remove a container
```
docker rm <container id>
```
#### docker remove image
``` 
docker rmi ubuntu
```
#### check docker images
``` 
docker images
```
#### cheeck docker info
```
docker info
```
